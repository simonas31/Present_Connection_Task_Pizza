## Present Connection Internship Task
Front-End: Vue.js\
Back-End: Laravel\
Database engine type: in-memory.\
Database example is `pizza_orders.sql` file.

## Installation
Launch xampp.\
Import `pizza_orders.sql` file to database.\
Run commands:\
`composer install`\
`npm install`

## Run
Run commands in two separate terminals:\
`npm run dev`\
`php artisan serve`
